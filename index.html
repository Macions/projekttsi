O egzaminie

Egzamin zawodowy INF.03 sprawdza umiejętności związane z tworzeniem i administrowaniem stronami internetowymi, aplikacjami webowymi oraz bazami danych. Składa się z części teoretycznej i praktycznej, a jego zdanie potwierdza uzyskanie kwalifikacji niezbędnych do pracy jako technik informatyk. Podczas egzaminu sprawdzane są m.in.:

  • projektowanie i obsługa relacyjnych baz danych,
  • wykorzystanie języka SQL do tworzenia i modyfikacji danych,
  • programowanie aplikacji internetowych z użyciem HTML, CSS, JavaScript i PHP,
  • obsługa formularzy i integracja z bazą danych,
  • konfiguracja i administrowanie środowiskiem serwerowym,
  • projektowanie interfejsów użytkownika, w tym układu strony, grafiki i elementów multimedialnych,
  • zapewnianie responsywności oraz dostępności stron internetowych,
  • łączenie warstwy front-end z back-endem w aplikacjach internetowych.

Egzamin INF.03 to nie tylko sprawdzian wiedzy, ale także praktycznych umiejętności potrzebnych w realnej pracy informatyka.

Etapy egzaminu

Część teoretyczna

Test pisemny obejmuje pytania jednokrotnego wyboru dotyczące teorii związanej z tworzeniem i administrowaniem stronami internetowymi, aplikacjami webowymi oraz bazami danych, a także BHP i multimediami.

Część teoretyczna trwa 60 minut i składa się z 40 pytań.

Część praktyczna

Wykonanie zadania praktycznego obejmującego:

  • projektowanie i implementację struktury strony internetowej z wykorzystaniem HTML i CSS,
  • tworzenie dynamicznych elementów strony za pomocą JavaScript,
  • budowanie i obsługę bazy danych w SQL,
  • integrację front-endu z back-endem przy użyciu PHP,
  • administrowanie środowiskiem serwerowym,
  • Wykonanie grafiki.

Całość zadania realizowana jest w czasie 150 minut i oceniana na podstawie jakości wykonania oraz zgodności kodu z poleceniem.

FAQ

Co mogę mieć ze sobą na egzaminie INF.03?
Na egzaminie INF.03 można mieć przy sobie długopis z czarnym tuszem oraz trzeba mieć dokument potwierdzający tożsamość.
Z jakich środowisk programistycznych mogę korzystać podczas egzaminu?
Dozwolone są środowiska wskazane przez komisję egzaminacyjną, takie jak Visual Studio Code, XAMPP, phpMyAdmin, Notepad++ czy Sublime Text. Wszystkie środowiska muszą być zainstalowane lokalnie na komputerze egzaminacyjnym i nie mogą mieć dostępu do internetu.
Czy mogę korzystać z internetu lub materiałów pomocniczych?
Nie – podczas egzaminu INF.03 korzystanie z internetu oraz materiałów pomocniczych, takich jak książki, notatki czy dokumentacja online, jest zabronione. Egzamin odbywa się w trybie offline.
Jak wygląda punktacja i ile trzeba zdobyć, żeby zdać?
Część teoretyczna trwa 60 minut i składa się z 40 pytań jednokrotnego wyboru. Aby zdać, należy uzyskać minimum 50% punktów.

Część praktyczna trwa 150 minut i polega na wykonaniu projektu strony internetowej, aplikacji webowej oraz zarządzaniu bazą danych. Aby zdać należy zdobyć co najmniej 75% punktów.
Czy można korzystać z własnych plików lub szablonów?
Nie, podczas egzaminu wszystkie pliki muszą być tworzone od podstaw. Gotowe szablony, kody źródłowe lub pliki zapisane wcześniej nie są dozwolone.
Jak przygotować się do części praktycznej w domu?
Warto ćwiczyć tworzenie stron internetowych z użyciem HTML, CSS, JavaScript, PHP oraz SQL, budowanie i zarządzanie bazami danych, instalowanie i konfigurację środowisk serwerowych lokalnie, testowanie responsywności i dostępności stron oraz dokumentowanie kodu i tworzenie instrukcji użytkownika. Regularne ćwiczenia w warunkach zbliżonych do egzaminacyjnych pomagają w opanowaniu wymaganych umiejętności.
Czy zadania są indywidualne czy zespołowe?
Zadania są indywidualne. Każdy zdający musi samodzielnie wykonać powierzone mu zadania praktyczne oraz odpowiedzieć na pytania teoretyczne. Współpraca z innymi osobami podczas egzaminu jest zabroniona.

HTML

HTML (HyperText Markup Language) to podstawowy język służący do tworzenia i strukturyzowania stron internetowych. Na egzaminie INF.03 znajomość HTML jest kluczowa – umożliwia tworzenie nagłówków, akapitów, list, linków, obrazów i innych elementów multimedialnych. Prawidłowe użycie HTML pozwala przygotować stronę, która może być dalej stylizowana za pomocą CSS i uzupełniana dynamicznymi funkcjami przy użyciu JavaScript oraz PHP. Opanowanie HTML jest fundamentem do realizacji zadań praktycznych egzaminu, w tym projektowania struktury strony i integracji z bazami danych.

HTML

CSS

CSS (Cascading Style Sheets) to język służący do stylizowania stron internetowych. Na egzaminie INF.03 znajomość CSS pozwala nadać stronie estetyczny wygląd, ustawić kolory, czcionki, marginesy, odstępy i pozycjonowanie elementów. Umiejętność tworzenia layoutów, responsywnych układów dla różnych urządzeń oraz efektów wizualnych jest ważna przy realizacji zadań praktycznych. CSS współpracuje z HTML, umożliwiając tworzenie atrakcyjnych i przejrzystych interfejsów użytkownika oraz przygotowanie strony do integracji z JavaScript i PHP.

CSS

JavaScript

JavaScript to język programowania, który pozwala tworzyć dynamiczne i interaktywne elementy stron internetowych. Na egzaminie INF.03 znajomość JavaScript umożliwia dodawanie funkcji takich jak walidacja formularzy, animacje, interakcje z użytkownikiem i dynamiczne zmiany treści strony. JavaScript współpracuje z HTML i CSS, a także może integrować się z PHP i bazami danych, co jest istotne przy wykonywaniu zadań praktycznych egzaminu, takich jak tworzenie w pełni funkcjonalnej aplikacji webowej.

JavaScript

PHP

PHP to język skryptowy używany do tworzenia dynamicznych aplikacji internetowych po stronie serwera. Na egzaminie INF.03 PHP umożliwia obsługę formularzy, przetwarzanie danych przesyłanych przez użytkownika, komunikację z bazami danych oraz generowanie treści HTML w zależności od potrzeb aplikacji. Umiejętność stosowania PHP w praktyce pozwala integrować front-end z back-endem, realizując funkcjonalne zadania egzaminacyjne, takie jak tworzenie prostych systemów zarządzania treścią czy interaktywnych stron internetowych.

PHP

SQL

SQL (Structured Query Language) to język używany do zarządzania relacyjnymi bazami danych. Na egzaminie INF.03 znajomość SQL pozwala tworzyć i modyfikować tabele, wstawiać, aktualizować i usuwać dane, a także wykonywać zapytania umożliwiające wyszukiwanie i sortowanie informacji. Umiejętność stosowania SQL jest niezbędna przy integracji baz danych z aplikacjami webowymi, obsłudze formularzy oraz realizacji zadań praktycznych, takich jak budowanie prostych systemów zarządzania danymi.

Multimedia

Multimedia w kontekście egzaminu INF.03 obejmują wykorzystywanie obrazów, grafiki, wideo, dźwięku oraz innych elementów wizualnych i interaktywnych na stronach internetowych. Niektóre zadania praktyczne wymagają także tworzenia własnej grafiki. Umiejętność prawidłowego wstawiania i formatowania grafik, dodawania wideo i audio oraz optymalizacji plików multimedialnych jest istotna przy tworzeniu estetycznych i funkcjonalnych stron. Dobrze przygotowana sekcja multimediów wpływa na atrakcyjność interfejsu użytkownika.
